# LINE Bot GAS 生成器 - 專案更新說明

## ✅ 完成的更新

### 1. 新增下載 GS 檔案功能
- 在生成結果頁面添加了「💾 下載 GS 檔案」按鈕
- 點擊後會自動下載生成的 `.gs` 檔案
- 檔案名稱格式：`LineBot_GAS_2025-10-15T12-30-45.gs`
- 包含時間戳記，方便管理多個版本

### 2. 建立 Netlify 部署資料夾
已創建 `netlify/` 資料夾，包含：
```
netlify/
├── index.html              # 靜態版本的主頁面（無 Flask 模板語法）
├── static/
│   ├── css/
│   │   └── style.css       # 樣式表
│   └── js/
│       └── script.js       # 前端邏輯（包含完整的 GAS 程式碼生成）
├── netlify.toml            # Netlify 配置檔
├── README.md               # 英文部署說明
└── 部署說明.md              # 中文快速部署指南
```

### 3. 前端程式碼生成
- 將原本 Python 後端的程式碼生成邏輯完全移到前端 JavaScript
- 使用者填寫表單後，直接在瀏覽器生成 GAS 程式碼
- 無需後端伺服器，完全靜態化
- API Keys 不會傳送到任何伺服器，保護隱私

## 🚀 如何部署到 Netlify

### 最簡單方式（推薦）：拖放部署

1. 打開瀏覽器，前往：https://app.netlify.com/drop
2. 將 `netlify` 資料夾直接拖放到頁面上
3. 等待上傳完成（約 10-20 秒）
4. 完成！網站已上線

### 自訂網域名稱為 line-ai-prompt

1. 部署完成後，點擊「Site settings」
2. 在「Site information」區塊，點擊「Change site name」
3. 輸入：`line-ai-prompt`
4. 保存後，網址變成：`https://line-ai-prompt.netlify.app`

### 使用命令列部署

```bash
cd netlify
netlify login
netlify deploy --prod
# 然後按照提示操作
```

## 🎯 兩個版本的差異

### Flask 版本（原始版本）
- 需要 Python 環境
- 需要執行 `python app.py`
- 適合本地開發和測試
- 位置：專案根目錄

### Netlify 版本（靜態版本）
- 純前端，無需後端
- 可部署到任何靜態網站託管服務
- 適合公開分享使用
- 位置：`netlify/` 資料夾

## 📝 使用建議

1. **本地開發測試**：使用 Flask 版本（`python app.py`）
2. **公開分享**：部署 Netlify 版本到 `https://line-ai-prompt.netlify.app`
