Part 2: 系統需求與環境建置

安裝 Python、Flask，啟動本地端生成器

Python 3.7+ Flask 虛擬環境

📋 系統需求

在開始之前，請確保你的電腦已具備：

🐍 Python 3.7+

本專案使用 Python 撰寫，需要 3.7 或更高版本。

下載 Python

📦 pip

Python 套件管理工具，通常隨 Python 一起安裝。

🌐 瀏覽器

Chrome、Edge、Firefox 等現代瀏覽器皆可。

📁 專案檔案

從 GitHub 下載本專案的完整檔案。

提示：不確定是否已安裝 Python？在命令列輸入 python --version 檢查！

📥 步驟一：下載專案

方法 A：使用 Git Clone（推薦）

git clone https://github.com/ChatGPT3a01/CH8-LINE-Bot-Generator.git
cd CH8-LINE-Bot-Generator

方法 B：下載 ZIP 檔案

  1. 前往 GitHub 專案頁面
  2. 點擊右上方的 Code 按鈕
  3. 選擇 Download ZIP
  4. 解壓縮到你想要的資料夾
完成後，你會看到 app.py、requirements.txt、templates/ 等檔案和資料夾。

🔧 步驟二：建立虛擬環境

虛擬環境讓專案的套件獨立管理，不影響系統其他 Python 專案。

Windows 使用者

# 方法一：雙擊執行 setup_venv.bat

# 方法二：手動執行
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

macOS / Linux 使用者

# 方法一：執行 setup_venv.sh

# 方法二：手動執行
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

🚀 步驟三：啟動應用程式

Windows 使用者

# 雙擊執行 start.bat
# 或手動執行：
venv\Scripts\activate
python app.py

macOS / Linux 使用者

# 執行 start.sh
# 或手動執行：
source venv/bin/activate
python app.py
啟動成功後，終端機會顯示：
Running on http://0.0.0.0:8806

✅ 啟動成功！

當你看到以下畫面，表示 Flask 伺服器已經準備就緒：

❓ 常見問題排解

找不到 python 命令？

確認 Python 已加入系統 PATH。重新安裝時勾選「Add Python to PATH」。

pip install 失敗？

試著升級 pip：
python -m pip install --upgrade pip

Port 8806 被佔用？

關閉其他使用此 port 的程式，或修改 app.py 中的 port 設定。

瀏覽器打不開？

試試 http://127.0.0.1:8806，或檢查防火牆設定。
