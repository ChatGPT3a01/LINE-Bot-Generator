LINE Token Google Sheets AI API Key

📝 系統基本設定：三大必填欄位

1️⃣ LINE Channel Access Token

Bot 的「身份證」，讓程式能代表你的 Bot 發送訊息。

取得方式：LINE Developers Console

2️⃣ Google 試算表 ID

知識庫的位置，存放問答資料的試算表 ID。

取得方式：試算表網址中 /d/ 和 /edit 之間

3️⃣ AI API Key

AI 服務的鑰匙，讓程式能呼叫 Gemini 或 OpenAI。

取得方式：Google AI Studio 或 OpenAI Platform

🔑 取得 LINE Channel Access Token

  1. 前往 LINE Developers Console
  2. 選擇你的 Provider（或建立新的）
  3. 建立或選擇一個 Messaging API Channel
  4. 進入「Messaging API」頁籤
  5. 找到「Channel access token」區塊
  6. 點擊「Issue」按鈕產生權杖
  7. 複製這串長長的英數字串
注意：Token 是機密資訊，請勿公開分享！

📊 建立知識庫試算表

在 Google Sheets 建立一個新試算表，格式如下：

問題 答案
營業時間是？ 我們的營業時間是週一至週五 9:00-18:00，週末休息。
如何聯絡客服？ 您可以撥打客服專線 02-1234-5678，或寄信到 service@example.com。
有提供外送服務嗎？ 是的，我們提供外送服務，滿 500 元免運費。

🧠 選擇 AI 服務

本工具支援兩種 AI 服務，你可以根據需求選擇：

Google Gemini（推薦）

  • 繁體中文理解能力出色
  • API 費用經濟實惠
  • 提供免費使用額度
  • 申請流程簡單

申請 Gemini API Key →

OpenAI GPT

  • 邏輯推理能力強
  • 創意生成出色
  • 英文對話表現佳
  • 技術文件生成優秀

申請 OpenAI API Key →

🔷 Gemini 模型選擇

模型 特色 適用場景
Gemini 2.5 Flash 快速、經濟 一般客服對話、快問快答
Gemini 2.5 Pro 理解力強、推理佳 複雜問答、專業諮詢
Gemini 3.0 Flash 新一代快速模型 高效能需求
Gemini 3.0 Pro 新一代專業模型 進階應用
建議：初次使用選擇 Gemini 2.5 Flash，效能與成本最平衡！

🟢 OpenAI 模型選擇

模型 特色 適用場景
GPT-4o 標準版，平衡效能 一般對話、客服
GPT-4.1 進階理解力 專業諮詢、技術支援
GPT-4-turbo 高速版本 需要快速回應
GPT-5.1 最新最強 高階應用
注意：OpenAI API 需要先儲值才能使用。

🔐 取得 Gemini API Key

  1. 前往 Google AI Studio
  2. 使用 Google 帳號登入
  3. 點擊「Create API Key」按鈕
  4. 選擇一個 Google Cloud 專案（或建立新專案）
  5. 幾秒鐘後 API Key 就會產生
  6. 複製這個 Key 貼到表單中
免費額度：Gemini 提供每月一定量的免費使用，對初學者很友善！
