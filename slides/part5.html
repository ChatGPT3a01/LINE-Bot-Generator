Chapter 8 LINE Bot Generator

Part 5: 程式碼生成與 GAS 部署

生成客製化程式碼，部署到 Google Apps Script

程式碼生成 Google Apps Script 部署

🚀 步驟一：生成程式碼

  1. 確認所有必填欄位都已填寫完成
  2. 檢查 LINE Token、試算表 ID、API Key
  3. 點擊「🚀 生成 GAS 程式碼」按鈕
  4. 等待 1-2 秒，程式碼會顯示在下方文字框

📋 步驟二：複製程式碼

生成完成後，你會看到完整的 JavaScript 程式碼：

// ===================================
// LINE Bot × Google Sheets × Gemini AI
// 由 LINE Bot GAS 生成器自動產生
// ===================================

const CHANNEL_ACCESS_TOKEN = '你的 LINE Token';
const GEMINI_API_KEY = '你的 API Key';
const SPREADSHEET_ID = '你的試算表 ID';
// ... 更多設定與邏輯 ...
點擊「📋 複製程式碼」按鈕，程式碼會複製到剪貼簿！

📝 步驟三：前往 Google Apps Script

1

開啟 Google Apps Script

前往 script.google.com

2

建立新專案

點擊左上角「新專案」按鈕

3

刪除範例程式碼

選取全部的預設程式碼並刪除

4

貼上生成的程式碼

按 Ctrl+V 貼上剛才複製的程式碼

🖥️ Google Apps Script 編輯器

儲存專案：點擊磁碟圖示（或 Ctrl+S），為專案取名如「LINE Bot 客服系統」

⚙️ 步驟四：部署設定

1

點擊「部署」→「新增部署作業」

2

選取類型：「網路應用程式」

3

填寫說明

例如：v1.0 - 初始版本

4

設定權限

  • 執行身分：「我」
  • 具有存取權的使用者：「所有人」

🔐 授權流程

首次部署時，系統會要求授權：

  1. 點擊「授權存取」
  2. 選擇你的 Google 帳號
  3. 點擊「進階」
  4. 點擊「前往『專案名稱』（不安全）」
  5. 點擊「允許」
為什麼顯示「不安全」？
因為這是你自己寫的程式，還沒經過 Google 官方驗證。但因為是你自己的程式，所以可以放心授權。

✅ 部署成功！

部署成功後，你會得到一個網址：

https://script.google.com/macros/s/AKfycbzXXXXX.../exec
這就是你的 Webhook URL！請務必複製下來。

⚠️ 重要提醒

網址結尾要有 /exec

確保複製的網址結尾是 /exec，不是 /dev

修改後要重新部署

修改程式碼後，需要「管理部署作業」→「編輯」→「新版本」

試算表權限

確保試算表的分享設定允許「知道連結的人可以檢視」

API Key 安全

程式碼中包含 API Key，請勿公開分享此專案

📋 Part 5 總結

本單元完成的步驟：

  1. 點擊「生成程式碼」按鈕
  2. 複製生成的 GAS 程式碼
  3. 前往 Google Apps Script 建立新專案
  4. 貼上程式碼並儲存
  5. 部署為網路應用程式
  6. 完成授權流程
  7. 取得 Webhook URL
