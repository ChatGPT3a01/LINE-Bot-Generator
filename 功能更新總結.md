## 🎯 三個下載按鈕的差異

### 1️⃣ 下載原版 GAS 檔案（黃色按鈕）
- **位置**：頁面頂部
- **顏色**：黃色（#FFC107）
- **檔案**：`LineBot_Sample_Original.gs`
- **內容**：基礎版本的 GAS 程式碼
- **使用時機**：想要最簡單的範本，自己手動填入參數

### 2️⃣ 複製程式碼（藍色按鈕）
- **位置**：生成結果區
- **顏色**：藍色（#4A90E2）
- **功能**：複製到剪貼簿
- **使用時機**：直接貼到 Google Apps Script 編輯器

### 3️⃣ 下載 GS 檔案（藍色按鈕）
- **位置**：生成結果區
- **顏色**：藍色（#4A90E2）
- **檔案**：`LineBot_GAS_[時間戳記].gs`
- **內容**：根據使用者設定客製化的 GAS 程式碼
- **使用時機**：想保存客製化的程式碼，方便備份和版本管理

## 🚀 使用流程建議

### 流程 A：快速上手（使用原版）
1. 點擊頁面頂部的「📥 下載原版 GAS 檔案」
2. 打開下載的 `LineBot_Sample_Original.gs`
3. 填入您的：
   - LINE Channel Access Token
   - Gemini API Key
   - Google 試算表 ID
4. 部署到 Google Apps Script
5. 完成！

### 流程 B：客製化設定
1. 填寫表單中的各項設定
2. 點擊「🚀 生成 GAS 程式碼」
3. 選擇以下任一方式：
   - 點擊「📋 複製程式碼」→ 貼到 GAS 編輯器
   - 點擊「💾 下載 GS 檔案」→ 保存備份後再上傳
4. 部署到 Google Apps Script
5. 完成！

## 🌐 部署到 Netlify

### 最簡單的方式
1. 前往 https://app.netlify.com/drop
2. 將 `netlify` 資料夾拖放到頁面
3. 等待上傳完成
4. 自訂網站名稱為 `line-ai-prompt`
5. 分享網址：`https://line-ai-prompt.netlify.app`

詳細步驟請參考：`netlify/部署說明.md`

## 💡 技術亮點

1. **雙版本支援**
   - Flask 版本：適合本地開發測試
   - Netlify 版本：適合公開分享使用
