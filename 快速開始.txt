========================================
  LINE Bot GAS 程式碼生成器
  快速開始指南
========================================

⚙️ 步驟一：建立虛擬環境並安裝套件（首次使用）
----------------------------
雙擊執行「setup_venv.bat」

這個腳本會自動：
1. 建立 Python 虛擬環境
2. 啟動虛擬環境
3. 安裝所有必要套件

※ 只需要執行一次！


🚀 步驟二：啟動應用程式（之後每次使用）
----------------------------
方法 1：雙擊「start.bat」檔案（推薦）

方法 2：在命令提示字元中手動執行：

    venv\Scripts\activate
    python app.py


🌐 步驟三：開啟瀏覽器
----------------------------
在瀏覽器網址列輸入以下任一網址：

    http://localhost:8806
    或
    http://127.0.0.1:8806


✅ 步驟四：填寫設定表單
----------------------------
1. 輸入 LINE Channel Access Token
2. 輸入 Google 試算表 ID
3. 選擇 AI 服務（Gemini 或 OpenAI）
4. 輸入對應的 API Key
5. 自訂機器人回答風格
6. 點選「生成 GAS 程式碼」


📋 步驟五：複製程式碼並部署
----------------------------
1. 複製生成的 GAS 程式碼
2. 前往 https://script.google.com
3. 建立新專案，貼上程式碼
4. 部署為網路應用程式
5. 將網址設定到 LINE Developers 的 Webhook URL


💡 提示
----------------------------
- 完整說明請參考 README.md
- 如需更多可自訂選項，請參考 LineBot_GAS_Customizable_Items.md


❓ 常見問題
----------------------------
Q: 啟動時出現「找不到模組」錯誤？
A: 請確認已執行 pip install -r requirements.txt

Q: 無法開啟網頁？
A: 確認 8806 埠未被其他程式佔用

Q: 生成的程式碼在哪裡？
A: 按下生成按鈕後，程式碼會顯示在網頁下方的文字框中


祝你使用愉快！🎉
