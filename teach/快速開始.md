# 🚀 快速開始 - 5 分鐘上手

## 最簡單的方式

### 步驟 1：開啟教學網頁

**Windows：**
1. 找到 `teach` 資料夾中的 `index.html`
2. 雙擊檔案，用瀏覽器開啟
3. 開始學習！

**Mac：**
1. 找到 `teach` 資料夾中的 `index.html`
2. 右鍵 → 打開方式 → Chrome/Firefox/Safari
3. 開始學習！

### 步驟 2：開始探索

頁面會自動載入，你會看到：

```
🤖 從零開始建立 LINE Bot
完整學習 HTML、CSS、JavaScript，打造專業的 LINE Bot GAS 生成器

[開始學習按鈕]
```

點擊「開始學習」即可！

## 學習流程

### 第一階段：基礎知識（2小時）

```
1. HTML 基礎 (30分鐘)
   ├── 認識 HTML 標籤
   ├── 學習表單元素
   └── 完成練習題

2. CSS 樣式 (45分鐘)
   ├── 了解 CSS 寫法
   ├── 互動調整工具 ⭐
   ├── Flexbox 排版
   └── 完成練習題

3. JavaScript (45分鐘)
   ├── 變數與函式
   ├── DOM 操作
   ├── 事件處理
   ├── 計數器實作 ⭐
   └── 待辦事項 ⭐
```

### 第二階段：專案實作（3小時）

```
4. 專案建置 (90分鐘)
   ├── 專案架構
   ├── HTML 結構
   ├── CSS 美化
   ├── JS 邏輯
   └── 部署流程

5. 實戰練習 (90分鐘)
   ├── 填寫表單
   ├── 生成程式碼 ⭐
   ├── 複製/下載
   └── 測試功能
```

## 互動功能導覽

### 🎨 CSS 調整工具（超推薦！）

在 CSS 基礎章節，你可以：

1. 選擇顏色 → 即時看到效果
2. 調整字體大小 → 立即更新
3. 修改內距、圓角 → 所見即所得

```
調整滑桿 → 看效果 → 查看生成的 CSS 程式碼
```

### 🧮 計數器示範

在 JavaScript 章節，親手做一個：

```
[  -  ]  [ 歸零 ]  [  +  ]
        ↓
      計數器
```

點擊按鈕，數字會變化！

### ✅ 待辦事項應用

實作一個迷你版 Todo App：

列表：

□ 學習 CSS

點擊事項可以刪除！

## 遇到問題？

### 程式碼沒有反應
1. 打開開發者工具（F12）
2. 查看 Console 是否有錯誤訊息
3. 檢查程式碼是否有拼寫錯誤

### 樣式沒有顯示
1. 確認 CSS 檔案路徑正確
2. 檢查 class 名稱是否一致
3. 查看瀏覽器相容性

### 功能無法運作
1. 確認 JavaScript 檔案有載入
2. 檢查事件監聽器是否正確
3. 查看 Console 錯誤訊息

## 小提示

### 💡 學習技巧
1. **動手做** - 不要只是看，要實際寫程式碼
2. **多實驗** - 修改範例程式碼，看看會發生什麼
3. **做筆記** - 記錄重要概念和常用語法
4. **問問題** - 不懂就問，到社群發問

### 🎯 效率學習
1. 每學完一個章節，休息 10 分鐘
2. 不要一次學太多，分段學習效果更好
3. 複習比一直往前學更重要
4. 實作專案是最好的學習方式

### 📱 行動學習
教學網頁支援手機瀏覽！
- 通勤時可以複習理論
- 搭配電腦實作更有效
