# 📚 LINE Bot GAS 生成器 - 互動式教學網頁

從零開始學習 HTML、CSS、JavaScript，並建置完整的 LINE Bot GAS 生成器專案！

## 🎯 教學目標

這是一個完整的互動式教學網頁，適合：

- ✅ 完全沒有程式基礎的初學者
- ✅ 想學習網頁前端技術的學生
- ✅ 希望建置 LINE Bot 的開發者
- ✅ 想了解專案實作流程的學習者

## 📖 課程內容

### 1. HTML 基礎 (30分鐘)
- 什麼是 HTML？

### 3. JavaScript 互動 (60分鐘)
- 變數與資料型別
- 函式與 DOM 操作
- 事件處理
- **互動式計數器示範**
- **待辦事項實作**

### 4. 專案建置 (90分鐘)
- 完整專案架構
- HTML 結構設計
- CSS 美化技巧
- JavaScript 邏輯實作
- 部署到 Netlify

### 5. 實戰練習 (自由時間)
- **實際操作表單生成器**
- 下載生成的 GAS 程式碼
- 學習進階功能

## 🚀 快速開始

### 方法 1：直接開啟（最簡單）

1. 用瀏覽器打開 `index.html`
2. 開始學習！

### 方法 2：使用本地伺服器（推薦）

如果你安裝了 Python：

```bash
# Python 3
cd teach
python -m http.server 8000

# 然後開啟瀏覽器訪問：
# http://localhost:8000
```

如果你安裝了 Node.js：

```bash
# 安裝 http-server（首次）
npm install -g http-server

# 啟動伺服器
cd teach
http-server

# 然後開啟瀏覽器訪問顯示的網址
```

### 方法 3：VS Code Live Server

1. 安裝 VS Code 擴充功能 "Live Server"
2. 在 `index.html` 上右鍵
3. 選擇 "Open with Live Server"
