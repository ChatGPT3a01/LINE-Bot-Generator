HTML 基礎知識

1. 什麼是 HTML？

HTML（HyperText Markup Language）是超文本標記語言，是構建網頁的骨架。 就像蓋房子需要先搭建骨架一樣，網頁也需要 HTML 來定義結構。 

<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <title>我的第一個網頁</title>
</head>
<body>
    <h1>歡迎來到我的網站！</h1>
    <p>這是我的第一個網頁。</p>
</body>
</html>

歡迎來到我的網站！

這是我的第一個網頁。

小提示： HTML 標籤通常成對出現，有開始標籤 <tag> 和結束標籤 </tag>

2. CSS 的三種寫法

1 行內樣式

<p style="color: red;">紅色文字</p>
不推薦：難以維護和重複使用

2 內部樣式

<head>
    <style>
        p { color: red; }
    </style>
</head>
適用：單頁應用或測試用

3 外部樣式 ⭐ 推薦

<head>
    <link rel="stylesheet" href="style.css">
</head>
/* style.css */
p { color: red; }
最佳實踐：易於維護、可重複使用

5. Flexbox 排版（現代必學）

Flexbox 是 CSS 中最強大的排版工具，我們的專案大量使用它：

.container {
    display: flex;              /* 啟用 Flexbox */
    justify-content: center;    /* 水平置中 */
    align-items: center;        /* 垂直置中 */
    gap: 20px;                  /* 子元素間距 */
}

.box {
    width: 100px;
    height: 100px;
    background: #06C755;
}
1
2
3
Flexbox 常用屬性：
  • justify-content: 主軸對齊（水平）
  • align-items: 交叉軸對齊（垂直）
  • flex-direction: 排列方向（row/column）
  • gap: 元素間距

2. 變數與資料型別

// 宣告變數的三種方式
let name = "阿亮老師";        // 可以改變的變數
const age = 30;               // 不可改變的常數
var city = "台北";            // 舊式寫法（不推薦）

// 常見資料型別
let text = "字串";            // String（字串）
let number = 42;              // Number（數字）
let isTrue = true;            // Boolean（布林值）
let items = [1, 2, 3];        // Array（陣列）
let person = {                // Object（物件）

};
最佳實踐： 優先使用 const，只有當變數需要改變時才使用 let

3. 函式（Function）

函式是可以重複使用的程式碼區塊：

// 一般函式
function greet(name) {
    return "你好，" + name + "！";
}

// 箭頭函式（現代寫法）
const greet = (name) => {
    return `你好，${name}！`;
};

// 呼叫函式
console.log(greet("小明"));  // 輸出：你好，小明！

綜合練習

建立一個簡單的待辦事項（Todo）功能：

  • 輸入框讓使用者輸入待辦事項
  • 「新增」按鈕將事項加入清單
  • 顯示所有待辦事項
  • 點擊事項可以刪除
    function addTodo() {
    const input = document.getElementById('todoInput');
    const list = document.getElementById('todoList');
    const text = input.value.trim();

    if (text === '') return;

    const li = document.createElement('li');
    li.textContent = text;
    li.onclick = function() {
        this.remove();
    };

    list.appendChild(li);
    input.value = '';
}

    步驟 3：JavaScript 邏輯

    最重要的部分 - 處理表單提交和生成程式碼：

    // 監聽表單提交事件
document.getElementById('configForm').addEventListener('submit', async (e) => {
    e.preventDefault();  // 阻止表單預設提交行為

    // 收集表單資料
    const formData = {
        channel_access_token: document.getElementById('channel_access_token').value,
        spreadsheet_id: document.getElementById('spreadsheet_id').value,
        ai_service: document.querySelector('input[name="ai_service"]:checked').value,
        // ... 其他欄位
    };

    // 生成 GAS 程式碼
    const gasCode = generateGasCode(formData);

    // 顯示結果
    document.getElementById('generated-code').value = gasCode;
    document.getElementById('result-section').style.display = 'block';
});

// 程式碼生成函式
function generateGasCode(config) {
    return `// LINE Bot GAS 程式碼
const CHANNEL_ACCESS_TOKEN = '${config.channel_access_token}';
const SPREADSHEET_ID = '${config.spreadsheet_id}';
// ... 更多程式碼
`;
}

// 複製按鈕功能
document.getElementById('copy-btn').addEventListener('click', () => {
    const code = document.getElementById('generated-code');
    code.select();
    document.execCommand('copy');
    alert('程式碼已複製！');
});

// 下載按鈕功能
document.getElementById('download-btn').addEventListener('click', () => {
    const code = document.getElementById('generated-code').value;
    const blob = new Blob([code], { type: 'text/plain' });
    const url = URL.createObjectURL(blob);
    const link = document.createElement('a');
    link.href = url;
    link.download = 'LineBot.gs';
    link.click();
    URL.revokeObjectURL(url);
});
