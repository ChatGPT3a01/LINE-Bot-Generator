════════════════════════════════════════════════════════════════
各檔案用途說明
════════════════════════════════════════════════════════════════

【核心程式】

app.py
  • Flask Web 應用主程式
  • 處理表單提交與 GAS 程式碼生成
  • 支援 Gemini 和 OpenAI 雙模式

requirements.txt
  • Python 套件相依清單
  • 目前只需要 Flask 3.0.0

.gitignore
  • Git 版本控制忽略規則
  • 排除 venv、__pycache__ 等


【網頁介面】

templates/index.html
  • 主頁面 HTML
  • 完整的表單介面
  • 包含作者與社群連結

static/css/style.css
  • 完整的 CSS 樣式
  • LINE 綠色主題設計
  • 響應式佈局

static/js/script.js
  • 前端互動邏輯
  • 表單驗證
  • AI 服務切換
  • 程式碼複製功能


【Windows 腳本】

setup_venv.bat
  • 自動建立 Python 虛擬環境
  • 安裝必要套件
  • 只需執行一次

start.bat
  • 啟動 Flask 應用
  • 自動啟動虛擬環境
  • 日常使用主要腳本

check_environment.bat
  • 檢查環境設定是否正確
  • 驗證 Python 版本
  • 檢查套件安裝狀態


【macOS/Linux 腳本】

setup_venv.sh
  • 功能同 setup_venv.bat
  • 適用於 Unix-like 系統

start.sh
  • 功能同 start.bat
  • 適用於 Unix-like 系統

check_environment.py
  • Python 版本的環境檢查
  • 跨平台使用
  • 提供詳細的檢查報告


【說明文件】

README.md
  • 最完整的專案說明文件
  • 包含安裝、使用、FAQ
  • 英文風格，適合分享

快速開始.txt
  • 最精簡的入門指南
  • 3 步驟快速上手
  • 適合初次使用者

使用流程.md
  • 詳細的操作流程圖解
  • 從安裝到部署的完整步驟
  • 包含疑難排解

專案說明.txt
  • 專案功能總覽
  • 技術架構說明
  • 使用情境範例

專案結構.txt
  • 本檔案
  • 詳細的檔案結構說明
  • 各檔案用途解釋

作者與社群資訊.md
  • 阿亮老師簡介
  • Facebook、YouTube 連結
  • 3A科技實驗室社團介紹
  • 如何參與學習社群

LineBot_GAS_Customizable_Items.md
  • GAS 可自訂項目清單
  • 包含系統設定、AI 風格等
  • 進階使用參考

常見問題
════════════════════════════════════════════════════════════════

Q: 需要安裝什麼軟體？
A: 只需要 Python 3.7+ 和 pip

Q: 如何更新專案？
A: 下載新版檔案，覆蓋原有檔案即可（venv 資料夾保留）

Q: 可以修改程式碼嗎？
A: 可以！所有程式碼都可以自由修改

Q: 遇到問題怎麼辦？
A: 1. 查看 README.md 的 FAQ
   2. 加入 3A科技實驗室社團提問
