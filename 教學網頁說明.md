## 🚀 如何使用

### 最簡單方式
```bash
# 直接開啟
cd teach
# 雙擊 index.html 即可
```

### 使用本地伺服器（推薦）
```bash
# Python 方式
cd teach
python -m http.server 8000
# 訪問 http://localhost:8000

# Node.js 方式
npx http-server teach
```

### VS Code Live Server
1. 安裝 Live Server 擴充功能
2. 右鍵 `index.html`
3. 選擇 "Open with Live Server"
