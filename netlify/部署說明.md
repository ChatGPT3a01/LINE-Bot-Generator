# 🚀 快速部署到 Netlify

## 最簡單的方式（推薦）

### 方法 1️⃣：拖放部署（30 秒完成）

1. 打開瀏覽器，前往 https://app.netlify.com/drop
2. 將整個 `netlify` 資料夾拖放到頁面上
3. 等待上傳完成（約 10-20 秒）
4. 完成！您的網站已上線

您會得到一個像這樣的網址：`https://random-name-123456.netlify.app`

#### 自訂網域名稱（可選）

1. 部署完成後，點擊「Site settings」
2. 在「Site information」區塊，點擊「Change site name」
3. 輸入：`line-ai-prompt`
4. 保存後，您的網址變成：`https://line-ai-prompt.netlify.app`

---

## 方法 2️⃣：使用命令列部署

如果您想用命令列，請在 `netlify` 資料夾中執行：

```bash
# 第一次部署
netlify deploy --prod

# 然後按照提示操作：
# 1. 選擇 "Create & configure a new project"
# 2. 選擇您的團隊
# 3. 輸入網站名稱：line-ai-prompt
# 4. 確認部署目錄：. (輸入一個點)
```

## ⚠️ 常見問題

### Q: 部署後網站樣式跑掉了？
A: 請確認 `static/css/style.css` 和 `static/js/script.js` 檔案都在正確的位置

### Q: 按鈕點擊沒反應？
A: 打開瀏覽器開發者工具（F12），查看 Console 是否有錯誤訊息

### Q: 想要更新網站內容？
A: 修改檔案後，再次拖放整個資料夾到 Netlify，或使用 `netlify deploy --prod`
